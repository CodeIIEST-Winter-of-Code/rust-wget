# rust-wget
rust-wget is a simple re-write of the famous wget utility in rust.

## Requirements
* Latest version of rust is preferred
        ```rustup update```
           
## Crates used
* clap 
* reqwest
* indicatif
* colored

## Usage

Compile the binary and you are set to use it.
